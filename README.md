## <a id="why"></a>Why SNS Workbench

SNS Workbench provides a professional, branded environment for developers. It is based on VSCodium to ensure:
- **No Telemetry**: All Microsoft tracking is removed.
- **Open Source**: Built from the MIT-licensed `vscode` source code.
- **Customized Experience**: Tailored branding and configuration for SNS groups.
