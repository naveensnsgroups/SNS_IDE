#  SNS Workbench - User & Installation Guide

Welcome to **SNS Workbench**, a powerful, branded IDE designed for modern development.

---

##  System Requirements

This version of SNS Workbench is optimized for Windows PCs and Laptops.

| Requirement | Specification |
| :--- | :--- |
| **Operating System** | Windows 10, Windows 11 (also supports Windows 7/8) |
| **Architecture** | 64-bit (x64) |
| **RAM** | 4GB Minimum (8GB Recommended) |
| **Disk Space** | ~500MB for installation |

---

##  Installation Instructions

Follow these simple steps to set up SNS Workbench on your computer:

1.  **Download**: Get the `SNS_Workbench_Setup.exe` from the official download link.
2.  **Run Installer**: Double-click the `.exe` file to start the setup.
3.  **Security Note**: If Windows shows a "SmartScreen" warning:
    *   Click **"More info"**.
    *   Click **"Run anyway"**.
4.  **Configuration**: 
    *   Follow the prompts to choose your installation folder.
    *   Check the box for **"Create a desktop icon"** if you want easy access.
5.  **Finish**: Click **Install** and then **Finish**.

---

##  Getting Started

Once installed, you can launch SNS Workbench from your Desktop or Start Menu. 

### Key Features:
